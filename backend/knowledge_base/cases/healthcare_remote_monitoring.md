## Problem

[PLACEHOLDER — fill with real project details]

A US-based healthcare startup needed to build a mobile application allowing patients with chronic conditions to monitor vital signs at home using Bluetooth-connected medical devices (pulse oximeters, blood pressure monitors, glucometers).

Key challenges:
- BLE (Bluetooth Low Energy) integration with 6 different hardware manufacturers
- HIPAA compliance for all patient data storage and transmission
- FDA software as a medical device (SaMD) guidelines
- Reliable data sync even on poor mobile connections

---

## Solution

[PLACEHOLDER — fill with real solution details]

Interexy built a React Native app with:
- Custom BLE layer using react-native-ble-plx supporting 6 device types
- End-to-end encryption for all health data
- Offline-capable data capture with background sync
- Clinician dashboard (React web) with alert thresholds and trend analysis
