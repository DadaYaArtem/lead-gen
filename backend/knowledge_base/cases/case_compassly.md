# Compassly

## Общая информация
- Клиент: Compassly / Crimson Heights
- Локация: USA (штат Юта)
- Отрасль: Healthcare, Special Needs, Autism
- Формат: Outsource
- Статус: Активный, продукт запущен
- Финансирование: правительство штата Юта

## Команда
- Разработка с нуля (from scratch)

## Стек
- Backend: Node.js
- Mobile: React Native
- Кроссплатформенная разработка

## Проблема клиента
Необходимо хранить всю медицинскую информацию в одном месте. Делаем интеграцию с 80 больницами. Личный кабинет для пациентов с аутизмом — через него они могут смотреть историю болезни.

## Решение
Разработали приложение для людей с аутизмом с нуля. Интеграция с 80 больницами, удобный time tracker для caregivers, система для отслеживания прогресса по навыкам и задачам.

## Ключевые фичи
- Интеграция с 80 больницами
- Личный кабинет для пациентов (история болезни)
- Time tracker для caregivers — отслеживание прогресса детей
- Задачи и навыки для детей с аутизмом
- Приложение для детей, но также подходит для взрослых с аутизмом

## Значимость
- Финансируется правительством штата Юта
- Social impact проект
- Healthcare + интеграция с больничными системами
