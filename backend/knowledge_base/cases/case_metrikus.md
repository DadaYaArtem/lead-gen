# Metrikus

## Общая информация
- Клиент: Metrikus
- Локация: United Kingdom
- Отрасль: Real Estate, Manufacturing, IoT, PropTech
- Формат: Outstaff
- Статус: Активный

## Команда
- Размер: 7 человек
- Заменили inhouse-команду клиента: клиент уволил inhouse и нанял Interexy
- Оценка проекта: $400K+

## Стек
- Outstaff формат — работа в стеке клиента

## Проблема клиента
Metrikus — платформа для эффективности зданий. Помогает организациям превращать хаос данных в понятную аналитику и визуализации. Нормализует данные из различных источников (сенсоры, BMS и т.д.), упрощая доступ через интерфейс или API. У клиента была inhouse-команда которая не справлялась.

## Решение
Interexy полностью заменила inhouse-команду. Команда из 7 человек работает над платформой в формате outstaff.

## Ключевые фичи
- Оптимизация пространства (Space optimization) — выявление неэффективно используемых площадей для сокращения затрат
- Продуктивность и благополучие (Productivity and wellbeing) — анализ качества воздуха (IAQ) для улучшения здоровья и комфорта сотрудников
- Устойчивость (Sustainable buildings) — мониторинг ESG-метрик для устойчивого развития

## Результаты
- Клиенты Metrikus: Siemens, Philips, крупные университеты
- 30-40% улучшение показателей эффективности зданий
