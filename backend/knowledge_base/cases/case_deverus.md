## Проблема клиента
Deverus — крупный провайдер background screening для HR-индустрии. Нужен редизайн legacy-портала, реализация цифрового кошелька на блокчейне для хранения результатов проверок, и AI-чатбот с ML для автоматизации потока заявок.

## Решение
Три направления: редизайн legacy web application, разработка независимого блокчейн-модуля (digital wallet на Polygon), интеграция AI-чатбота с ML для извлечения данных из загружаемых документов.

## Ключевые фичи
- Digital Wallet на блокчейне (Polygon) с данными background screening внутри токена
- AI чатбот с ML-моделями для извлечения данных из документов
- Редизайн legacy-портала
- Валидация background report через блокчейн
