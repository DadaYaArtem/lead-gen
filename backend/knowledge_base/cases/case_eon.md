## Проблема клиента
E.ON нуждается в проектировании и оптимизации потоков данных для отчётности и аналитики. Работа с большими данными, интеграция с внешними системами, обеспечение надёжности ETL-процессов. Компания активно внедряет AI и облачные решения для оптимизации энергетических процессов.

## Решение
Interexy предоставила Data Engineer со специализацией в Data Vault для работы над data engineering инфраструктурой.

## Ключевые задачи
- Проектирование и оптимизация потоков данных для отчётности и аналитики
- Работа с Data Lakes / Data Warehouses
- Интеграция данных с внешними системами и обеспечение надёжности ETL
- Поддержка и развитие инфраструктуры для аналитики и BI
