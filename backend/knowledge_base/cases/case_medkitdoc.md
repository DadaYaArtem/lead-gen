# MedKitDoc

## Общая информация
- Клиент: MedKitDoc
- Локация: Германия (DACH)
- Отрасль: Healthcare, Telemedicine, B2B
- Формат: Outsource
- Статус: Завершён, приложение в AppStore

## Команда
- Размер: 6 человек
- Состав: 2 devs, 1 PM, 1 BA, 1 QA, 1 UI/UX
- Разработка: mobile development from scratch (iOS)

## Стек
- iOS: Swift
- Backend: PHP, Firebase
- Интеграции: Bluetooth LE (медицинские устройства)

## Проблема клиента
Телемедицинский проект для немецкого рынка. Клиент привлёк 2 миллиона евро для продолжения работы над back office решением. Сервис — телемедицинский продукт для профессионалов, которые постоянно в разъездах, а также для бизнес-подразделений, которым нужна быстрая медицинская помощь онлайн. Главный UX-вызов — глубокая интеграция с hardware (Bluetooth-устройства).

## Решение
Разработали iOS-приложение с нуля. Ключевая особенность — интеграция с MedKit (набор медицинских приспособлений: оксиметры, стетоскопы, тонометры), которые соединяются с приложением через Bluetooth. Реализовали видео- и аудио-звонки между пациентом и врачом, синхронное отражение данных с устройств.

## Ключевые фичи
- Bluetooth-интеграция медицинских устройств (оксиметры, стетоскопы, тонометры)
- Видео- и аудиоконсультации врач-пациент
- Синхронное отражение данных с устройств в реальном времени
- Hardware workflows для работы с MedKit
