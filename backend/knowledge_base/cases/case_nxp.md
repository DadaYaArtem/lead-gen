# NXP Semiconductors

## Общая информация
- Клиент: NXP Semiconductors N.V.
- Локация: Нидерланды (Eindhoven), онбординг в Гамбурге
- Отрасль: Полупроводники, автомобильная электроника, IoT, микроконтроллеры
- Формат: Outstaff
- Статус: Активный
- Масштаб клиента: один из крупнейших производителей полупроводников в Европе, выделена из Philips в 2006

## Команда
- Размер: 1 Project Manager
- Онбординг: выезд в Гамбург

## Проблема клиента
NXP разрабатывает микроконтроллеры и системы для автомобильной электроники (ADAS — беспилотные авто), управления двигателем, информационно-развлекательных систем. Также работает в IoT, мобильных технологиях (NFC для бесконтактных платежей), безопасности (чипы для паспортов и ID-карт).

## Решение
Interexy предоставила Project Manager для управления проектами разработки ПО и аппаратных решений.

## Ключевые задачи
- Управление проектами разработки software и hardware
- Координация команд разработки, тестирования и QA
- Планирование сроков, ресурсов и бюджетов
- Подготовка отчётности для руководства

## Значимость
- Глобальный enterprise-клиент (ex-Philips)
- Автомобильная электроника + IoT + NFC
- Подтверждение экспертизы Interexy на уровне крупнейших hardware-компаний мира
